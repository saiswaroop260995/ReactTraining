HTML:

Hyper Text Markup Language 

-> to describe web pages.

CSS:  Used to provide the design and layout for HTML Pages.


* W3C gave us HTML mainly used to create web pages.

Mark up Language:

A mark up language is a set of tags.

A tag is a keyword enclosed in an angular bracket.

ex: <html>


* How html pages gets rendered?

-> 

HTML Page to browser 

browser -> Creates DOM Tree and represents each element as an object.



Render Engine -> Follows DOM and renders the web page.

DOM Viewer:  http://software.hixie.ch/utilities/js/live-dom-viewer/



ex:

3. If we do not use ending tag, Browser adds it when it starts rendering.

4. DOM is responsible for dynamic web pages.

to be closed.

There are two types of elements in HTML.

1. block level elements : Any element that starts in a new line and occupies
the max width
2. inline : any element that starts in same line.

ex: span, a

ex:

<html>
    <head>

    </head>
    <body>

        <h1 style="background-color: red">This is heading 1</h1>
